# Basic for Super

super() 는 기본적으로 원래 두개의 param 을 가지고 있음.
첫번째 param 은 subclass.
두번째 param 은 첫 param 의 instance 인 object
python3 부터는 super() == super(Subclass, object) 와 동일함.
